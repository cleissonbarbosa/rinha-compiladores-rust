⚠️ **Notice**

**<span style="color:red">This project is not finished yet, so it is not recommended to use it in production.</span>**

## Run

### Install Rust

- [Rust](https://www.rust-lang.org/)

### Run project

```bash
cargo run --bin main ./examples/fib.rinha
```

### Run REPL
```bash
cargo run --bin repl
```
